# IPL Auction Simulator - Web Dashboard 📊

This is the frontend component of the IPL Auction Simulator, built with Next.js and TypeScript. It provides a visual interface for managing auctions, simulating matches, and analyzing team performance.

## ✨ Features

- **ESPN-Style UI**: High-density sports dashboard with a dark theme.
- **Live Simulator View**: Real-time visualization of match progress.
- **Analytics Hub**: Interactive charts and tables for player/team statistics.
- **Responsive Layout**: Designed for seamless viewing across devices.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🛠️ Built With

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Vanilla CSS
- **Fonts**: Inter, Outfit

## 📁 Structure

- `/app`: Routing and page components (Home, Simulator, Analytics, Tournament).
- `/components`: Shared UI elements (Navbar, Cards, Progress Bars).
- `/styles`: Global CSS and theme variables.
